# Litwave 1.1.0 — Initial Release

Litwave synchronizes smartphone flashlights and screens across any crowd — no Wi-Fi, no Bluetooth, no setup. Every phone stays in sync automatically using the clock already in your pocket.

## Features

- **Flash signal** — activate a morse-code flash pattern on your screen and flashlight with a single tap
- **Flashlight toggle** — sync the device torch to the morse stream with configurable delay offset
- **Signal timing calibration** — fine-tune screen fade and flashlight delay in Settings → Signal timing, with a built-in test pattern
- **Events** — create named events with scheduled start times, share via QR code or deep link
- **Deep links** — scan a QR code or open a link on any device to import and activate an event instantly
- **Custom messages** — type any message; unsupported characters are flagged before saving
- **Works fully offline** — no accounts, no server, no internet required
- **Auto-pause on background** — signal pauses when you lock your screen and recalculates on resume

## Things to Know

- Flashlight auto-stops after 10 minutes to protect the LED
- Signal timing calibration is device-dependent — defaults work for most devices but may need adjustment on yours
